Foghorn.Data.Comp.Anncomp

Description

This module defines the variable set for Compustat’s ExecuComp Annual Compensation (ANNCOMP) database, represented as a GADT (AnncompV). Each constructor corresponds to a named variable and is indexed by an EconType (e.g., Float_, Int_, Text_, Date_).
